# data_raw

This folder is reserved for raw or externally sourced input data.

For this project, raw sequencing files are not redistributed in the repository.

The replication uses publicly accessible processed data from:

- `curatedMetagenomicData`
- `curatedMetagenomicAnalyses`

These resources provide standardized abundance profiles and metadata for public colorectal cancer microbiome cohorts.

If needed, the original paper also references public raw-data accessions for several cohorts. Those accession links should be obtained from the original publication and related public repositories.

This folder is intentionally kept empty in the GitHub repository.
